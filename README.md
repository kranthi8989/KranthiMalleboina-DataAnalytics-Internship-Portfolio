#  Project Overview

This repository showcases the work completed during my **Data Analytics Internship**. Throughout the internship, I worked on an end-to-end data analytics project using the **Cafe Sales Dataset**, applying data cleaning, exploratory data analysis (EDA), dashboard development, data storytelling, and portfolio creation.

The project demonstrates my ability to transform raw data into meaningful business insights using industry-standard data analytics tools and techniques.
