# Task 3 - Dashboard Development

## Objective
Develop an interactive dashboard to visualize key business metrics and provide meaningful insights from the Cafe Sales dataset.

## Activities Performed
- Imported the cleaned Cafe Sales dataset.
- Designed an interactive dashboard to monitor business performance.
- Created charts to visualize sales trends and product performance.
- Analyzed category-wise sales and revenue distribution.
- Displayed key performance indicators (KPIs) for better decision-making.
- Organized the dashboard with clear and user-friendly visual elements.

## Dashboard Features
- Total Sales Overview
- Revenue Analysis
- Product-wise Sales
- Category-wise Performance
- Sales Trend Analysis
- Customer Insights
- Key Performance Indicators (KPIs)

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab
- Microsoft PowerPoint

## Key Insights
- Identified top-performing products.
- Compared sales across different categories.
- Visualized revenue trends.
- Highlighted important business metrics through an interactive dashboard.
- Supported data-driven decision-making using visual analytics.

## Deliverables
- Dashboard
- Dashboard Screenshots
- Dashboard Report
