# Task 2 - Exploratory Data Analysis (EDA)

## Objective
Analyze the cleaned Cafe Sales dataset to identify trends, patterns, and valuable business insights using exploratory data analysis techniques.

## Activities Performed
- Loaded the cleaned dataset.
- Explored the dataset using descriptive statistics.
- Analyzed sales performance.
- Identified top-selling products.
- Examined category-wise sales.
- Analyzed customer purchasing patterns.
- Created visualizations to understand data trends.

## Visualizations Created
- Bar Chart
- Line Chart
- Pie Chart
- Histogram
- Box Plot
- Heatmap

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Identified sales trends and product performance.
- Compared category-wise sales.
- Analyzed revenue distribution.
- Generated business insights from visualizations.

## Deliverables
- EDA Notebook
- Data Visualization Charts
- EDA Report
